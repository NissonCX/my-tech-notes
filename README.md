# my-tech-notes

技术问答笔记仓库，记录日常学习中提出的问题和解答。

## 目录结构

- `Java/` - Java 基础、JVM、并发等
- `AI/` - LangChain4j、RAG、向量数据库等
- `Others/` - 其他分类（按需新建）

## 笔记格式

```markdown
# 标题

日期：YYYY-MM-DD
标签：标签1, 标签2

## 问题
原问题内容

## 回答
核心回答内容

## 相关问题
- 拓展问题 1 → 回答
- 拓展问题 2 → 回答

## 技术拓展
延伸知识点（可选）
```

## 使用原则

- 同类问题合并记录在同一文档
- 分类文件夹按需创建
- 图片跟随笔记文件同目录
